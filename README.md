# StudyMate Agent

AI-powered personal exam preparation agent.

## Quick Start

```bash
npm install
npm run build
./dist/cli.js init
./dist/cli.js ingest ./materials/sample.pdf
./dist/cli.js plan --exam 2026-09-15 --daily 60
./dist/cli.js today
./dist/cli.js quiz
./dist/cli.js grade --answers answers.json
```
